CSS Selectors

When using CSS, selectors help you target specific elements in your HTML code. They can help you change the design, font, color, and placement of the elements on your page.

The "background-color" selector was used to change the background color of this section of the page

The font can be changed using "font-family". You can use different fonts for any element, although with web pages, the maximum number of fonts you want to use is 3.

You can also change your font to be bold, italic, or underlined, using the "text-decoration" syntax.

You can target more than one element using selectors as well. For example, I could select both elements in this area by using the syntax: "div p { background-color: purple; }"

The grouping selector allows you to change all elements of the same type with one selector.

The Box Model

With CSS, the "Box Model" refers to the design and layout of a web page. It refers to the model that every HTML element can be contained in a box, consisting of the margin,border,padding and content.

This box showcases the box model. The padding is set at 10px, the margin is set at 50px for top and bottom and 30px for left and right, with a solid black border, and the text and image inside this box is the content.

Positions

Static Positioning

The content in this div is static. The "static" position is the default. It's not affected by top, bottom, right or left properties.

Absolute Positioning

With absolute positioning, an element is positioned absolutely to its first positioned parent, like the text in this div.

Relative Positioning

This div has the "relative" position. Any element with relative positioning is positioned relative to its normal position.

Fixed Positioning

Fixed positioning is used to keep an element in the same position, regardless of scrolling. It was used for the navigation bar of this page.

Floats

Similar to positioning, Float is used to position and format elements of a web page.

This image was positioned using "float:left"

This image was positioned using "float:right"

With "float:none", the image will remain exactly where it is placed in the text.

With "clear", this property specifies which elements next to the cleared element can float, and on which side.

The Clearfix Hack

When an element is bigger than the element containing it, there can be an overflow, causing elements on a page to lose its aesthetic value. By using "overflow: auto;", it can fix this issue.
